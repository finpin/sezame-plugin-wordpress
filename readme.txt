Passwordless multi-factor authentication.

== Description ==

Unlike password-based solutions that require you to remember just another PIN or password, sezame is a secure and simple multi-factor authentication solution.
You only need the username and your fingerprint on your smartphone to log into any sezame-enabled site.


== Installation ==

Upload the Sezame plugin to your blog, activate it, go to the Sezame menu in the admin area and complete the setup process.

== Frequently Asked Questions ==

= Does this plugin work with woocommerce? =

Yes, it detects woocommerce automatically and enables fingerprint authentication for woocommerce consumers.
